En el proyecto de pruebas unitarias deben cambiar las rutas del archivo de la base de datos xml,
Tambien deben cambiar las rutas al acceso de la data de las pruebas, sino no se ejecutaran las pruebas
[------OJO----------]

En cada constructor en las pruebas unitarias deben cambiar la ruta donde usted quiera que se crea la base de datos.

Y para que los métodos de prueba obtengan datos para la realización de pruebas deben cambiar la ruta....En la carpeta Data se encuentra el archivo donde
estan los datos de prueba
